# Product Landing Page

A modern, fully responsive, and optimized landing page built using semantic HTML5 and modular SCSS.

---

## 🚀 Live Demo

* **Live Link:** [View Live Site](https://evgeniy45.github.io/landing-page/)
* **Project Repository:** [GitHub Repo](https://github.com/Evgeniy45/landing-page)

---

## 🛠️ Tech Stack & Features

* **HTML5 & CSS3** – Semantic layout and clean design.
* **SASS (SCSS)** – Modular styles using modern `@use` rules.
* **BEM Methodology** – Scalable and maintainable naming convention.
* **Vite** – Fast bundling and asset optimization.
* **Responsive & Pixel Perfect** – Fully adapted for all devices, from mobile to desktop.

---

## 💻 Local Setup

To run this project locally, execute the following commands in your terminal:

```bash
# Clone the repository
git clone [https://github.com/Evgeniy45/landing-page.git](https://github.com/Evgeniy45/landing-page.git)

# Navigate into the project folder
cd landing-page

# Install dependencies
npm install

# Start the development server
npm start
