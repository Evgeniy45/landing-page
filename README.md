# Nothing Products - Landing Page

A responsive and modern landing page created for "Nothing" brand products. This project was built to demonstrate the practical application of modern HTML and CSS layout techniques. It is fully adaptive and provides a seamless user experience across mobile, tablet, and desktop devices.

## Links
- [Live Demo (GitHub Pages)](https://Evgeniy45.github.io/layout_landing-page/)
- [Figma Design Mockup](https://www.figma.com/file/DtkQmQ797hk0nI4KfMi2Uq/BOSE-New-Version?type=design&node-id=6802-139&t=L7eKz5YKLN0m5WxR-0)

## 🛠️ Technologies Used
- **HTML5** (Semantic markup, accessibility features)
- **SCSS / Sass** (Variables, mixins, nesting)
- **CSS Grid & Flexbox** (Advanced responsive layouts)
- **BEM Methodology** (Component-driven architecture and class naming)
- **Stylelint** (Code quality and formatting consistency)

## How to run locally

To run this project on your local machine, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Evgeniy45/layout_landing-page.git
   ```

2. Navigate to the project folder:
   ```bash
   cd layout_landing-page
   ```

3. Install dependencies (for SCSS compilation and Stylelint):
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```
